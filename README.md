## Potato64: a disk space analyzer
Potato64 is an Open-Source Windows disk space analyzer that will tell you what is eating space in your drive. 

## How to use it?
1. Download the .zip file.
2. Extract it anywhere on your computer.
3. Double-click Potato64.exe.
4. After it starts, point it at a folder and click Scan.
